Choosing a package

Semantic is designed to be used as part of a front-end build system like Grunt or Gulp so that you can modify themes, site-specific variables, and UI overrides during development.

Components

Select All Elements

Button
A button indicates a possible user action
Divider
A divider visually segments content into groups
Flag
A flag is is used to represent a political state
Header
A header provides a short summary of content
Icon
An icon is a glyph used to respresent something else
Image
An image is a graphic representation of something
Input
An input is a field used to elicit a response from a user
Label
A label displays content classification
List
A list is used to group related content
Loader
A loader alerts a user to wait for an activity to complete
Rail
A rail is used to show accompanying content outside the boundaries of the main view of a site
Reveal
A reveal displays additional content in place of previous content when activated
Segment
A segment is used to create a grouping of related content
Step
A step shows the completion status of an activity in a series of activities

Select All Collections

Breadcrumb
A breadcrumb is used to show heirarchy between content
Form
A form displays a set of related user input fields in a structured way
Grid
A grid is used to harmonize negative space in a layout
Menu
A menu displays grouped navigation actions
Message
A message displays information that explains nearby content
Table
A table displays a collections of data grouped into rows

Select All Views

Advertisement
An ad displays third-party promotional content
Card
A card displays site content in a manner similar to a playing card
Comment
A comment displays user feedback to site content
Feed
A feed presents user activity chronologically
Item
An item view presents large collections of site content for display
Statistic
A statistic emphasizes the current value of an attribute

Select All Modules

API
API allows elements to trigger actions on a server
Accordion
An accordion allows users to toggle the display of sections of content
Checkbox
A checkbox allows a user to select a value from a small set of options, often binary
Dimmer
A dimmer hides distractions to focus attention on particular content
Dropdown
A dropdown allows a user to select a value from a series of options
Form Validation
A form validation behavior checks data against a set of criteria before passing it along to the server
Modal
A modal displays content that temporarily blocks interactions with the main view of a site
Nag
A nag is an important message that persists until dismissed by a user
Popup
A popup displays additional information on top of a page
Progress
A progress bar shows the progression of a task
Rating
A rating indicates user interest in content
Search
A search module allows a user to query for results from a selection of data
Shape
A shape is a three dimensional object displayed on a two dimensional plane
Sidebar
A sidebar hides additional content beside a page.
Sticky
Sticky content fixes itself to the browser viewport as content is scrolled
Tab
A tab is a hidden section of content activated by a menu
Transition
A transition is an animation usually used to move content in or out of view
Video
A video displays moving images with playback controls
Visibility
Visibility provides a set of callbacks for when a content appears in the viewport
